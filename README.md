# 🏃‍♀️ Maratona de Boston 2025 – Análise Exploratória de Dados

Este projeto é uma análise detalhada da base de dados da Maratona de Boston 2025, com foco em explorar o perfil dos participantes, desempenho por gênero e faixa etária, além de visualizar as diferenças entre elite e demais atletas.

## 🔍 Principais insights
- A maioria dos corredores está entre 35–49 anos
- Mulheres têm maior participação nas faixas jovens, homens predominam nas mais velhas
- A elite masculina chega a correr a 20 km/h, o dobro da média dos amadores

## 📊 Habilidades aplicadas
- Python, pandas, seaborn, matplotlib
- Estatística descritiva
- Criação de gráficos e interpretação de dados
- Estruturação de projeto de dados
